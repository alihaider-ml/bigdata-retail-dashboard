# In-Store Shopping Analytics

**Interactive dark-themed dashboard for visualizing customer behavior in retail using Big Data.**

---

## Overview
This project is a Big Data Analytics case study on the **in-store shopping experience**. It integrates three main datasets:

1. **Mobile App Data** – User browsing, clicks, and session data.  
2. **In-Store Transaction Data** – Product purchases, quantities, and revenue.  
3. **Geolocation / Movement Data** – Indoor trajectories and dwell times.

The dashboard helps visualize customer behavior, store traffic, and purchase patterns, enabling **data-driven retail optimization**.

---

## Features
- Minimalistic landing page with tagline.  
- Interactive sections for problem, datasets, and pipeline.  
- Cards and accordions for digestible content.  
- Dark theme with gradients, shadows, and hover effects.  
- Scroll-triggered fade-in animations.
